### 1. 문자열 메소드

- Python에서는 문자열(string)을 따옴표로 감싸서 표현하며, 다양한 내장 메소드를 제공합니다.
- 주요 메소드:
    - `split()`: 지정된 문자(열)을 기준으로 문자열을 분리
    - `replace()`: 특정 문자(열)을 다른 문자(열)로 대체
    - 특수 문자 및 대소문자 처리는 데이터 전처리 과정에서 중요함

### 2. 정규식(Regular Expression)

- 정규식은 특정 패턴을 가진 문자열을 검색, 치환, 추출하는 기법입니다.
- Python에서는 `re` 모듈을 사용하여 정규식을 구현합니다.

#### 정규식 기본 패턴

- `[]`: 대괄호 안의 문자 중 하나와 매치 (예: `[a-z]`는 소문자 알파벳)
- `+`: 앞의 패턴이 한 번 이상 반복
- `\w`: 알파벳, 숫자, 언더스코어(단어를 구성하는 문자)
- `\W`: `\w`가 아닌 모든 문자
- `\d`: 숫자 문자
- `\D`: 숫자가 아닌 모든 문자
- `\s`: 공백 문자
- `\S`: 공백이 아닌 모든 문자
- `^`: 문자열의 시작
- `$`: 문자열의 끝
- `.`: 임의의 한 문자
- `{n}`: 앞의 패턴이 정확히 n번 반복
- `{n,m}`: 앞의 패턴이 n~m번 반복

#### 주요 re 함수

- `re.compile()`: 정규식 패턴 컴파일
- `re.match()`: 문자열의 시작부터 패턴과 일치하는지 확인
- `re.search()`: 문자열 전체에서 패턴과 일치하는 부분 검색
- `re.findall()`: 패턴과 일치하는 모든 부분을 리스트로 반환
- `re.finditer()`: 패턴과 일치하는 모든 부분을 반복 가능한 객체로 반환
- `re.sub()`: 패턴과 일치하는 부분을 다른 문자열로 치환

python복사

import re

# 정규식 패턴 컴파일

pattern = re.compile(r'[a-zA-Z]+')

# 패턴과 일치하는 모든 부분 찾기

result = pattern.findall("ABIT 한글 123 emoji")

### 1. 예외(Exception)의 개념

- **예외**: 프로그램이 의도한 대로 동작하지 않는 경우
- **파이썬의 예외 종류**:
    - **문법 오류(Syntax Error)**: 코드 작성 문제로 컴파일 전에 발생하는 오류
    - **런타임 에러(Runtime Error)**: 프로그램 실행 중에 발생하는 오류

### 2. 예외 처리 방법

- **try-except 구문**:
    
    python복사
    
    try:
    
        # 예외가 발생할 가능성이 있는 코드
    
    except:
    
        # 예외 발생 시 실행할 코드
    
- **try-except-else-finally 구문**:
    
    python복사
    
    try:
    
        # 예외가 발생할 가능성이 있는 코드
    
    except:
    
        # 예외 발생 시 실행할 코드
    
    else:
    
        # 예외가 발생하지 않을 때 실행할 코드
    
    finally:
    
        # 예외 발생 여부와 관계없이 항상 실행할 코드
    

### 3. 예외 종류별 처리

- **특정 예외만 처리**:
    
    python복사
    
    try:
    
        # 코드
    
    except ValueError:
    
        # ValueError 발생 시 처리
    
- **여러 예외 동시에 처리**:
    
    python복사
    
    try:
    
        # 코드
    
    except (ValueError, EOFError):
    
        # ValueError나 EOFError 발생 시 처리
    
- **다양한 예외 개별 처리**:
    
    python복사
    
    try:
    
        # 코드
    
    except ValueError:
    
        # ValueError 처리
    
    except EOFError:
    
        # EOFError 처리
    
    except Exception:
    
        # 그 외 모든 예외 처리
    

### 4. 예외 정보 활용

- **except as 구문**:
    
    python복사
    
    try:
    
        # 코드
    
    except Exception as e:
    
        print(e)  # 예외 메시지 출력

### 6. 객체 지향 프로그래밍(OOP)

- **클래스 정의**:
    
    python복사
    
    class Car:
    
        # 클래스 변수
    
        total_num = 0
    
        name = "연지중고차"
    
        # 생성자 함수
    
        def __init__(self, name="입력 없음"):
    
            self.name = name
    
        # 클래스 메서드
    
        @classmethod
    
        def check(cls):
    
            print(f"이 회사 이름은 {cls.name}입니다")
    
        # 인스턴스 메서드
    
        def instance_check(self):
    
            print(f"이 instance 이름은 {self.name}입니다")
    
- **인스턴스 생성 및 사용**:
    
    python복사
    
    morning = Car()
    
    tesla = Car()
    
    morning.name = "모닝"
    
    tesla.name = "테슬라"

### 3. 파일 처리시 예외 처리

python복사

try:

    f = open("test.txt", "r")

except FileNotFoundError:

    f = open("test.txt", "w")

    f.write("새로 만든 파일입니다")

else:

    # 파일이 존재하는 경우 실행

    line = f.readline()

    while line:

        print(line)

        line = f.readline()

finally:

    f.close()  # 성공하든 실패하든 파일은 항상 닫아야 함

### 4. 사용자 정의 예외 예시

python복사

class NegativeDivisionError(Exception):

    pass

def positive_divide(a, b):

    if b < 0:

        raise NegativeDivisionError("음수로는 나눌 수 없습니다")

    return a / b

try:

    result = positive_divide(10, -2)

except ZeroDivisionError:

    print("0으로는 나눌 수 없습니다")

except NegativeDivisionError as e:

    print(e)

except Exception:

    print("알 수 없는 오류가 발생했습니다")
