### 1. 파이썬 데이터 타입의 특징

- 파이썬은 다른 언어와 달리 90년대에 만들어져 메모리 관리가 유연함
- 변수 선언 시 데이터 타입을 명시적으로 지정할 필요 없음
- 인터프리터가 자동으로 데이터 타입을 인식하고 지정함 (동적 타이핑)

### 2. 조건문

#### IF 문

- 파이썬은 if와 match-case 두 가지 조건문 제공
- 기본 구조: `if 조건: 실행문`
- 추가 조건은 `elif`로, 그 외 모든 경우는 `else`로 처리

#### 다중 조건문 작성 시 주의사항

- 실행문이 매번 평가되는 문제
    
    python복사
    
    # 비효율적인 방법
    
    if input("값을 입력하세요: ") == "1": ...
    
    elif input("값을 입력하세요: ") == "2": ...
    
    # 효율적인 방법
    
    value = input("값을 입력하세요: ")
    
    if value == "1": ...
    
    elif value == "2": ...
    

#### 논리 연산자

- 파이썬에서는 `and`, `or`, `not` 키워드 사용 권장
- 비트 연산자(`&`, `|`, `~`)는 논리 연산에 사용하지 않는 것이 좋음
- Short Circuit Evaluation 주의
    
    python복사
    
    # True or 뒤의 조건은 평가하지 않음
    
    if True or 조건: ...
    

#### 연산자 우선순위 주의

python복사

# 의도: 3 또는 2 또는 5가 arr에 있는지 확인

if 3 or 2 or 5 in arr: ...  # 잘못된 방법 (항상 True)

if 3 in arr or 2 in arr or 5 in arr: ...  # 올바른 방법

#### 중첩 조건문

- 너무 깊은 중첩은 코드 가독성 저하
- 중첩 깊이가 3 이상일 경우 플래그 변수를 사용해 중첩 줄이기

#### Match-Case문

- Python 3.10부터 도입된 문법
- 특정 값과 정확히 일치하는 경우에 사용하기 적합
- 직관적인 코드 작성 가능

python복사

match ans:

    case "Y" | "y":

        print("좋군요")

    case _:

        print("그렇군요")

#### 언패킹과 활용

python복사

match dinner.split():

    case ["피자", *others]:

        print("피자를 먹자", others)

    case _:

        print("다른걸 먹자")

#### Walrus 연산자 `:=`

- 할당과 동시에 값을 사용할 수 있는 연산자

python복사

match (dinner := dinner.split()):

    case ["피자", *others]:

        print("피자를 먹자", *others)

    case _:

        print("다른걸 먹자")

#### 삼항 연산자

- 한 줄로 if-else문 작성 가능

python복사

print("참") if 3 in arr else print("거짓")

### 3. 반복문

#### while 문

- 횟수가 정해지지 않은 반복에 주로 사용
- 기본 구조: `while 조건: 실행문`
- 반복 종료 조건을 반드시 포함해야 함

#### while문 작성 패턴

1. 초기값 설정
2. 조건 판별
3. 반복문 내에서 조건 변경 (종료조건 생성)

python복사

n = 1

while n <= 10:

    print(n)

    n += 1

#### break와 continue

- `break`: 반복문을 완전히 종료
- `continue`: 현재 반복을 건너뛰고 다음 반복으로 이동
- 주의: `break` 이후의 코드는 실행되지 않음

#### 효율적인 코드 작성

- 코드 줄 수 최소화
- 불필요한 연산 제거
- 적절한 초기값 설정으로 반복 횟수 줄이기

python복사

# 비효율적

n = 0

while n <= 10:

    if n % 2 == 1:

        print(n)

    n += 1

# 효율적

n = 1

while n <= 10:

    print(n)

    n += 2

### 1. For문

- **기본 구조**: `for 변수 in iterable_객체: 실행할 코드`
- **특징**: 횟수가 정해져 있는 반복에 사용
- **활용**: 집합자료형(리스트, 튜플, 셋, 딕셔너리) 또는 range와 함께 주로 사용
- **Iterable 객체**: 반복될 수 있는 객체(반복 가능 객체)로, `__iter__` 메서드를 가진 객체
- **공갈문자 관례**:
    - 단수/복수 규칙: 집합 자료형은 복수형(prices), 원소는 단수형(price)
    - 의미 없는 변수: `_` (언더바) 사용
    - 중첩 반복문: 바깥쪽은 i, 안쪽은 j, k 순으로 사용

### 2. Range 함수

- **사용법**: `range(시작값, 끝값+1, 간격)`
- **특징**: 메모리 효율적인 수열 생성 (리스트보다 메모리 사용량 적음)
- **예시**: `range(1, 10, 3)`은 1, 4, 7 생성

### 3. 집합자료형과 반복문

- **리스트/튜플**: 인덱스로 접근 가능한 순서가 있는 자료형
- **딕셔너리**:
    - 기본 for문에서는 키만 반복
    - `items()`: 키-값 쌍을 튜플로 반환 (언패킹 가능)
    - `keys()`: 키만 반환
    - `values()`: 값만 반환
- **셋**: 순서가 없으나 `__iter__` 메서드로 반복 가능

### 4. 반복문 제어

- **break**: 반복문 완전히 종료
- **continue**: 현재 반복 건너뛰고 다음 반복으로
- **조건문과 결합**: 반복문 안에 조건문 사용하여 특정 조건에 맞는 경우만 처리

### 5. 중첩 반복문

- **구조**: 반복문 안에 또 다른 반복문 작성
- **실행 순서**: 내부 반복문이 모두 실행된 후 외부 반복문의 다음 단계로 진행
- **권장사항**: 중첩은 3개 이하로 제한 (가독성/유지보수 위해)

### 6. 함수(Function)

- **정의**: `def 함수명(파라미터들): 실행할 코드`
- **목적**: 코드 재사용성 향상, 모듈화
- **특징**: 최소한의 동작 단위로 작성 권장 (15-20줄 내외)
- **리프 함수**: 더 이상 쪼갤 수 없는 단위의 함수
- **템플릿 함수**: 다른 함수들의 순서를 정해 실행하는 함수

### 7. 함수 종류

1. **입력값/리턴값 없는 함수**: `def 함수명(): 실행문`
2. **입력값만 있는 함수**: `def 함수명(파라미터): 실행문`
3. **리턴값만 있는 함수**: `def 함수명(): return 값`
4. **입력값과 리턴값 모두 있는 함수**: `def 함수명(파라미터): return 값`

### 8. 함수 파라미터와 인수

- **파라미터(매개변수)**: 함수 정의시 지정하는 변수명
- **인수(argument)**: 함수 호출시 실제 전달되는 값
- **기본값 파라미터**: `def 함수명(파라미터=기본값): 실행문`
- **파라미터 구분**:
    - `/` 앞의 파라미터: 위치 인자(positional argument)로만 사용 가능
    - `*` 뒤의 파라미터: 키워드 인자(keyword argument)로만 사용 가능

### 9. 독스트링(Docstring)

- **목적**: 함수 설명 문서화
- **작성법**: 함수 정의 바로 다음에 삼중 따옴표로 설명 작성
- **활용**: 함수 이름만 입력하면 독스트링 출력됨

### 10. return 문

- **특징**: 함수 종료와 동시에 값 반환 (break와 유사)
- **주의점**: return 이후 코드는 실행되지 않음

1. **구구단 출력 예시**:

python복사

# for문으로 구구단 출력

dan = int(input("단수를 입력하세요: "))

for i in range(1, 10):

i = 1

while i < 10:

2. **중첩 반복문으로 구구단 전체 출력**:

python복사

for dan in range(2, 10):

    print(dan, "단")

    for i in range(1, 10):

        print(dan, "x", i, "=", dan * i)

    print()  # 단 구분용 빈 줄

3. **딕셔너리 반복문 활용**:

python복사

fruits = {"딸기": "strawberry", "바나나": "banana", "사과": "apple", "포도": "grape"}

# 키만 반복

for fruit in fruits:

    print(fruit)

# 키-값 쌍 반복

for k, v in fruits.items():

    print(k, v)

# 값만 반복

for v in fruits.values():

    print(v)

4. **함수 정의와 호출 예시**:

python복사

# 입력값과 리턴값 모두 있는 함수

def add(value1, value2=0):

    """입력받은 두 값을 더해서 return하는 함수입니다."""
